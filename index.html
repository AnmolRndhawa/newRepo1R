git init - to make github to track this folder

git status - to check the status of changes

git add . - to add all the changes at once into local device

git commit -m "comment" - to save all the changes into git

git remote add origin LINK OF REPO - to add the github repo location link to our localfile

git remote -v - to verify the github repo locatcation

git push origin main - to put all the local files into Remote Github repo

git push -u origin main - to put all the local files into Remote Github repo - no need to add origin main again and again
